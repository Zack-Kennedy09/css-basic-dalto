*box-shadow (sombras)


Esta propiedad nos permite darle sombra a la caja
El primer parametro le damos tamaño al eje X = 2px,
el segundo al eje Y = 4px,
el tercero va a ser el tamaño de desenfoque = 15px,
el cuarto el tamaño del borde, generalmente se le da cero = 0,
y el quinto es el color = #000 (negro)
